  • Project2: Storing and Quering YAGO: Project Description
    As a short description, initially I downloaded the YAGO2 core and converter, and unzipped them. I could use the converter tool to load YAGO into Jena TDB Store and then query the ontology, but the tool also allows to query YAGO - either in the native query language or in SPARQL. So I chose the latler one and used the converter tool to query YAGO. I entered the SPARQL queries and got the results. You can check out the queries and results in here.

  • Project3: Mapping Database into (Semantic) Web using Triplify.
    To be consice I downloaded and installed the MySQL Database, and then downloaded the sample database called Sakila database and set it up on MySQL. I also installed Triplify. For mapping the database to semantic web we need to create a Configuration file for Triplify. Evetually, I ran the Triplify script in the web application, the rest is done by Triplify automatically, it creates a RDF File which consists of the database schema based on the queries that I had included in the configuration file.
