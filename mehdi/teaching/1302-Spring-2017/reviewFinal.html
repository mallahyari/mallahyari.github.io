Please note: Although there will not be direct questions from the topics studied for the midterm exam, you should study them since they are related to the topics mentioned for the final exam.

Review the following material in preparation for the final exam (some of the topics will be discussed in class this week):

  1. Abstract Data Types and collections
    • Abstract Data Type
    • Importance of data hiding
    • Collection as an object holding other objects
    • Data Structure
  2. Generic classes
    • Why are they useful?
    • Instantiating a generic class
    • Autoboxing for primitive types
  3. Stacks and Queues
    • Stack ADT
    • Queue ADT
    • Examples of stack and queue applications
    • Types of implementations (array-based and linked-based)
  4. Linked structures and List ADT
    • Self-referential class
    • Linked structure
    • Types of lists
    • Traversal of a linked list
    • Adding and removing elements (link modifications)
    • Bi-directional linked list
  5. Analysis of algorithms
    • Algorithm efficiency
    • Problem size and growth function
    • Time vs. space complexity
    • Asymptotic complexity
    • Big-Oh notation (definition)
    • Major Bi-Oh categories
  6. Recursion
    • Recursive definition
    • Base case and recursive case
    • Recursive problem solving
    • Examples of recursive definitions
    • Recursive method
    • Recursion vs iteration
  7. Searching and Sorting
    • Comparable vs Comparator interface
    • Generic methods
    • Generic type parameter constraints
      • The meaning of <T extends Comparable<T>>
      • The meaning of <T extends Comparable<? super T>>
    • Linear search
    • Binary search
    • Selection sort
    • Insertion sort
    • Bubble sort
    • QuickSort
    • MergeSort
    • Trace operation of the above on sample data
    • Average- and worst-case complexity, be able to explain and justify
  8. Trees
    • Definition
    • Root vs internal node vs leaf
    • Subtree
    • Path in a tree
    • Ancestor vs descendant
    • Node level, tree height, path length
    • Order of a tree, binary tree
    • Balanced tree, full tree, complete tree
    • Types of tree implementations -- linked tree implementation
    • Tree traversals (pre-, in-, post-, and level-order)
    • Binary Tree ADT
    • Examples of tree applications

The test will be a collection of questions, where you will have to:

  • answer multiple choice questions
  • provide a short answer or explanation
  • write a small fragment of code (not a complete program)
  • predict the behavior (execution outcome) of a small fragment of code
