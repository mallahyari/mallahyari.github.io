Project 1: Linux and Java warm-up

Due:  August 29, 2016

In this project, you will create, compile and run a simple Java program on our Linux server (nike).

Project description:

  1. Create a directory called cs1302 in your home directory on nike.  You will use this directory to implement projects in this course.  In cs1302, create a sub-directory Project1. In Project1, create two sub-directories src and classes. You should place your Java program (described below) in the src directory and your compiled files in the classes directory.
  2. Design and implement a Java class called User, which accepts only two command-line arguments and prints out a line of text of the form:

    3. Welcome your_firstname your_lastname to the nike server!

    For example:

    Welcome JOHN SMITH to the nike server!

    Your program must not output anything else. 

  3. If the number of command-line arguments is not 2 (greater/less), the following line must be printed out and program must end:

    Error! The number of arguments must be two.
    4. You should use one of the text editors available on nike to create the source code for your class.  Also, you should use the javac and java commands to compile and run your program, respectively.

  4. The part of the required output line after “Welcome ” should be the value returned from a call to the static method getUserInfo(), provided by the UserIdentifier class.  The method is declared as:

    5. public static String getUserInfo(String, String)

    Your program should simply print the string  ”Welcome ”  concatenated  with the value returned by the getUserInfo method concatenated with the string ”to the nike server!”. getUserInfo method receives the two command-line arguments from the user and returns the uppercase format of the string.

  5. The UserIdentifier class is included in a jar file, called ident.jar, which is available in directory /home/profs/mehdi/libs on nike (the directory and the jar file are publicly readable).  You will have to define the CLASSPATH environment variable, or use an appropriate option when compiling and running your program, in order to gain access to ident.jar.

Submit your Project1 directory to cs1302b on nike. You must use the submit command to submit your work, as shown below:

$ submit Project1 cs1302b

Important: You should execute the submit command while being in the parent directory of Project1, i.e. cs1302 (the current directory must be cs1302), or use the absolute path name to Project1.

Things to note:

  • Follow good coding style (proper variable names, indentation, etc).
  • Include comments in your program.
