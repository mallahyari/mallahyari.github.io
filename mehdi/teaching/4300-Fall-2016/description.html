CSCI 4300 - Web Programming
Fall 2016

Short Description

This course will primarily focus will be on client-side and server-side techniques for use on the World Wide Web. Interactive, dynamically generated, and database-enabled web pages are discussed. Course content changes frequently to incorporate new Internet technologies.

Course Obejctives:

At the end of the semester, all students will be able to do the following: define and use terms relevant to web development; recognize and interpret HTTP response codes; author HTML documents conforming to language standards, promulgated by the World-Wide Web Consortium; design and develop web applications incorporating dynamically generated pages; design and develop database-enable web applications; handle error conditions in web applications; create HTML documents that change structure in response to user or asynchronous events.

Topical Outline
  • How the web works?
  • HTML and Cascading Style sheets
  • JavaServer Pages, JavaBeans & Java Servlets
  • Working with databases
  • Model-View-Controller development
  • Database design and Web application
  • Database programming
  • Web application development on servers
  • Client-side scripting: JavaScript
  • Web services and Service-Oriented Architecture

Course Prerequisite: CSCI 1302. If you have not taken this course, you must see me immediately. I will assume that you have a good understanding of the material covered in 1302 at least at a B level. It is your responsibility to make up any deficiencies.
