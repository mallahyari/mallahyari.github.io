Project description:

Your application must have an HTML file called movie.html. This html file has an HTML form. The user should be able to use the HTML form to specify the their items of interest (i.e. actors, directors or movies) they need to get information about. You may use lists, text fields, and buttons as form elements. After the form has been submitted by the user, the application should retrieve the information about the selected item from the imdb database, and return the response HTML page with a listing of all information about the selected item.

In case a user enters an unknown term, the application should return an error page with a suitable error message.

It is your responsibility to discover the column names and create a suitable SQL query to retrieve the required data. For example, you may use the mysql command to connect to the imdb database and then discover the table names, as well as the defined columns and their types.

Your application must utilize a Java servlet and an HTML form.  The application’s WAR archive must have a name of the form firstname_lastname_imdb.war and it must be deployed to our Wildfly Application Server on wp420 by copying the WAR file to the directory /opt/wildfly/standalone/deployments/project2.  Remember to set the read permissions of the WAR file to world-readable.  In order to set the permissions, you need to run the following Unix command after you copied the file into the project2 directory.

chmod a+x /opt/wildfly/standalone/deployments/project2/firstname_lastname_imdb.war

Your application must be accessible at the following URL:

http://wp420.cs.uga.edu:8080/firstname_lastname_imdb/movie.html

(of course, substitute your name in the URL above).

You have to use your username and password that were given in the class to able to connect to our server. The command to connect to our server is: 

ssh your_username@wp420.cs.uga.edu

Important Notice: Before you create (export) your .war file and deploy it on the server, please make sure you have changed the username and password to the correct ones for the database imdb on the server.

How to use MySQL database

  • You require to download and install MySQL available at here.
  • You can connect to the MySQL database using the default username/password (root/given_password): 
    mysql -uroot -pgiven_password
  • Create a database and call it imdb.
  • Import the this imdb database to your newly created imdb database.
  • Check to make sure the imdb database has been populated successfully.
